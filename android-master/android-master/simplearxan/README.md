## Overview

此專案為測試Android兩產品(A4A、EIT)的動態Guards用

## Usage

1. 修改app/build.gradle內的EIT路徑

1. 若需測A4A，請將新bulid的apk重新進行A4A防護

1. 要進行A4A防護的話請自行修改A4A_APKProtection下的runA4A.sh內的參數設定。
